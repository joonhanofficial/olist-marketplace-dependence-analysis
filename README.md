# Olist Marketplace Dependence Analysis

A Power BI dashboard built to assess whether Olist’s marketplace growth was broad-based or shaped by visible seller and category concentration.

## Business Question
Is Olist’s marketplace growth broadly distributed, or meaningfully dependent on a narrow set of revenue drivers?

## Why It Matters
Top-line growth can look strong while still relying on a narrow set of sellers, categories, or activity drivers. This project was built to distinguish between broad-based marketplace participation and visible concentration.

## Project Scope
This analysis uses the public Olist Brazilian ecommerce dataset to evaluate marketplace performance through three main lenses:
- seller concentration
- category contribution
- revenue and order activity over time

## Tools
Power BI, DAX, Data Modeling

## Key Insights
- Top 10 sellers contributed **13.15%** of total revenue, showing meaningful but non-extreme seller concentration
- The top category contributed **9.26%** of total revenue, indicating visible category leadership without overwhelming dominance
- Revenue and order trends rose together over time, suggesting growth was supported by broader marketplace activity
- Overall, marketplace growth appeared broad enough to show meaningful participation, while still shaped by leading sellers and categories
